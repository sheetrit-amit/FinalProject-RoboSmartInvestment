Stage 1 — User input
Chat interface
User types a natural language investment query: “I want to invest $10,000 with medium risk”

Stage 2 — Intent extraction   LLM call #1
OpenRouter · free model
LLM parses the message and returns three structured fields:
balance number  ·  currency string  ·  risk_level Low / Med-Low / Medium / Med-High / High
{ balance, currency, risk_level }
3
Stage 3 — Candidate fetch
BigQuery · companies_risk_ratings
Queries companies_risk_ratings table with risk_level as the filter key. Returns 15–25 candidate ticker symbols whose risk rating matches the user's profile.
15–25 candidate tickers
4
Stage 4 — Technical scanner   Pre-filter gate

Each ticker scored on 11 momentum & trend signals from 6 months of daily data:

filtered tickers + technical_score[ ]
5
Stage 5 — Markowitz optimization
SciPy SLSQP · mean-variance · efficient frontier
Downloads historical daily prices from BigQuery (daily_prices). Computes expected returns + covariance matrix, then runs SciPy SLSQP to maximize the Sharpe Ratio on the Efficient Frontier.
Output: optimal weight per ticker — Σ weights = 1.0
{ ticker, weight }[ ] — Σ = 1.0
6
Stage 6 — Fundamental score fetch
BigQuery · ticker_grades
For each portfolio ticker, queries ticker_grades for pre-computed analyst-style scores (0–100).
Scores were generated by LLM analysis of SEC 10-Q filings. Returns: fundamental_score + explanation per ticker.
{ ticker, fundamental_score, explanation }[ ]
7
Stage 7 — Portfolio labelling
Rule-based classification (weight × fundamental score)
Each position assigned a label based on its portfolio weight and fundamental score:
weight >15% & score ≥70 → “Top Pick”
weight >15% & score <50 → “Statistical Hold”
weight ≤15% & score ≥70 → “Growth Opportunity”
everything else → “Balanced Position”
portfolio with labels assigned
8
Stage 8 — Narrative synthesis   LLM call #2
OpenRouter · free model
LLM receives the enriched portfolio + user context and writes a human-readable investment thesis.
Inputs: tickers[ ]weights[ ]fundamental_scores[ ]technical_scores[ ]labels[ ]risk_levelbalancecurrency
Output: narrative investment thesis text
narrative text
